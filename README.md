# 🎮 English Quest Web

Um jogo web interativo desenvolvido para ajudar estudantes a praticar a tradução e a pronúncia da língua inglesa. O projeto utiliza mecânicas de gamificação, como pontos de experiência (XP) e sistema de vidas, para tornar a aprendizagem mais envolvente.

## ✨ Funcionalidades

* **Modo de Tradução (Teclado):** O utilizador digita a tradução da palavra ou frase para português.
* **Modo de Pronúncia (Microfone):** O utilizador fala a palavra em inglês. O navegador reconhece a voz e valida a pronúncia.
* **Sistema de Áudio:** Leitura das palavras em inglês com sotaque nativo através de Text-to-Speech (TTS).
* **Gamificação:** Sistema de vidas (HP) e recompensas em XP (bónus para quem usar o microfone ou acertar frases completas).
* **Integração de API:** Tradução em tempo real utilizando a API pública MyMemory.

## 🛠️ Tecnologias Utilizadas

* **Back-end:** Python 3, Django
* **Front-end:** HTML5, CSS3, JavaScript
* **APIs & Bibliotecas:**
  * `requests` (para comunicação com a API de tradução)
  * *Web Speech API* (para reconhecimento de voz e síntese de fala no navegador)
